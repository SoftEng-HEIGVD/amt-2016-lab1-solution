get  /users

Get all users.

You can use the fieldName and fieldValue to filter the list of users by username, firstname or lastname. The fieldValue is used to retrieve all the user with the value contained in the specified field.

Request

Query Parameters

Name
Type
Constraints
Description
Example
Field name
string

Field name used to filter the data.

When used, should be set with fieldValue.

fieldName=username

Field value
string

Field value used to filter the data.

When used, should be set with fieldName.

fieldValue=hdup
